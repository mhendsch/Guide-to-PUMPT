PUMPT Introduction¶

The Platform for User Model Personalized Textbooks (PUMPT), integrates data from multiple sources, including e-textbooks and other learning tools. It uses this data to create an Open Learner Model (OLM). An OLM allows for a learner to self-reflect and self-monitor by showing them their learning progress in an easy-to-understand format.

This text uses PUMPT to create these web pages, as well as allow for questions and learner modeling.

For a more in-depth explanation, see [KK21] at CEUR for an introduction to the pub/sub server.

[KK21]

Kay, Judy and Bob Kummerfeld. PUMPT: An e-Textbook Platform Based on a Personal User Model for Learning. CEUR, 2021.
